# ITELECT4 Project - GT1 Part 2

## Project Concept
This application serves as a standard academic portal submission manager where students can enroll in courses, submit their GitHub repository links for project tasks, and track their scores and evaluation lifecycles.

## Core Structures & Types Defined
- **Interfaces**: `User`, `Course`, `Submission`, and `ApiResponse<T>` (Generic)
- **Type Aliases**: `ID`, `Coordinate`, `Formatter`, `StringOrNumber`, `Status`, `StudentWithCourse`, `UserUpdate`, and `NewSubmissionPayload`
- **Enums**: `SubmissionStatus` (Regular) and `UserRole` (Const)
